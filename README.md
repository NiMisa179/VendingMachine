# 1. Description
## Vending Machine
The simpler form of a vending machine. The user types the price of the product and the amount of money they insert. The output is the change in coins.

# 2. Features
* Proccesses decimal numbers
* User inserts 2 inputs. The price of the product they like to buy and the amount of money they inserted into the vending machine.
* Output provided based on inputs

# 3. Technologies
* Python3
* Decimal

# 4. Execute the program
1. Save the script as:
   vendingMachine.py
2. Run the program:
   python vendingMachine.py
